# Repo Standards

## Commit Standards

```
<tag>(scope): my commit message subject

my commit message body

---

my commit message footer
```

### Valid Commit Tags

- `build` - 
- `chore` -
- `docs` -  
- `feat` - 
- `refactor` - 
- `style` -

### Commit Scopes

The commit scope should match a folder name in one of the PNPM workspaces. Every folder name should be unique.
